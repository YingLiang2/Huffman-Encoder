To run the huffman encoding project. You must first have all the files in the same directory.
Next have the command prompt go to that directory.

The default filename is "hamlet.txt". Please change the filename to the textfile that you will encode
in the Main class, by the variable fileName.

Finally to run the code type into the command prompt: node.exe Main.js

Then you should get a "hamlet.txt.huff" file which is an encoded file of the hamlet.txt

To run the test type into the command prompt: node.exe Tests.js
